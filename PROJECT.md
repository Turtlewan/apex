# APEX — Coding/Development System

## Vision
APEX is a system of Claude Code skills that turns every AI coding session into a structured, reproducible workflow. It covers the full dev lifecycle — planning specs, executing builds, debugging, reviewing, and producing handoff docs — through a set of modular skills that route work to the right specialist context and preserve state across session boundaries.

## Users
Just me — a solo developer using Claude Code as a primary development partner.

## Success looks like
- Every session starts with full context in under 30 seconds
- Planning sessions produce actionable specs that coding sessions can execute without clarification
- No work is lost between sessions
- The skill set is self-improving: gaps get specced, built, and shipped through the same system it governs
