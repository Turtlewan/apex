# APEX — Coding/Development System

A system of AI-assisted development skills for Claude Code that structures every session into planning, building, debugging, and handoff — with state that survives across session boundaries.

## Skill locations
- Skills: `~/.claude/skills/apex-*/`
- Registry: `~/.claude/registry/`
- Templates: `~/.claude/skills/apex-orchestrate/templates/`

## Project docs
- `PROJECT.md` — vision and goals
- `REQUIREMENTS.md` — scope
- `ROADMAP.md` — phase plan
- `docs/status.md` — live state (specs, in-flight work, open questions)
- `docs/changes/` — pending specs for coding mode to execute
- `docs/handoff/` — session summaries
